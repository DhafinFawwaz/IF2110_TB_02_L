<h2 id="how-to-run">Cara Menggunakan Program</h2>
Gunakan wsl. Pastikan gcc dan makefile sudah terinstall.

### Compile & Run (Instant run)
```bash
make
```

### Compile Program
```bash
make build
```

### Run Program
```bash
make run
```

### Clean Program
```bash
make clean
```

### Run All Unit Test
```bash
make test
```

### Run Specific Unit Test
```bash
make test ADT/ListDinamis
```
